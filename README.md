## 📖 About The Project (عن المشروع)

**English:**
An interactive, mobile-first web platform designed to simplify and summarize the 156-page theoretical manual for the **Software Modeling and Analysis** course. The platform provides a smart study environment featuring bilingual explanations (English terms with Arabic clarifications), an interactive Flashcards system for memorizing engineering concepts, end-of-chapter quizzes, and structured placeholders for architectural diagrams.

**العربية:**
منصة ويب تعليمية تفاعلية (مصممة خصيصاً للهواتف الذكية) تهدف إلى تبسيط وتلخيص الملزمة النظرية (156 صفحة) لمادة **"نمذجة وتحليل البرمجيات"**. توفر المنصة بيئة دراسية ذكية تحتوي على شروحات مزدوجة اللغة (المصطلح بالإنجليزي والتوضيح بالعربي)، نظام بطاقات تعليمية (Flashcards) لحفظ المصطلحات الهندسية، اختبارات قصيرة نهاية كل فصل، وأماكن مخصصة لإضافة المخططات المعمارية.

## ✨ Features (المميزات)

* 📱 **Mobile-First & Dark Mode:** A modern, clean UI with a responsive design and an animated starry background for comfortable night studying.
* 🗂️ **Comprehensive Content:** Covers all 15 weeks of the theoretical syllabus efficiently using space-saving Accordion menus.
* 🃏 **Interactive Flashcards:** Over 60 smart flashcards to practice and memorize key software engineering terms and definitions.
* ✅ **Mini-Quizzes:** Built-in multiple-choice quizzes (36+ questions) with instant feedback and scoring to simulate exam patterns.
* 🖼️ **Diagram Placeholders:** Neatly designed placeholders allowing students to easily map and insert complex UML and system diagrams from the original PDF.
* 🌐 **Bilingual Learning:** Preserves original English terminology for exam readiness while providing clear Arabic explanations to ensure deep understanding.

---

## 🛠️ Built With (التقنيات المستخدمة)

This platform was built from scratch without relying on heavy frameworks, focusing purely on core front-end technologies:
* **HTML5** (Semantic structuring)
* **CSS3** (Advanced styling, Flexbox, 3D Transforms for Flashcards, Keyframe animations)
* **Vanilla JavaScript** (Dynamic content rendering, DOM manipulation, Quiz logic, Array-based data structuring)
